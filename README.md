## Project setup

```
tnpm install
```

### Compiles and hot-reloads for development

```
tnpm run serve
```

### Compiles and minifies for production

```
tnpm run build:prd // for production
tnpm run build:stg // for staging
```

### Lints and fixes files

```
tnpm run lint
```

### Unit test

```
tnpm run test:unit
```
